# User Stories

## Epic: Compute Pricing Comparison

### Story 1
As a user, I want to enter vCPU, memory, region, OS, and monthly hours so that I can compare compute costs.

### Story 2
As a user, I want AWS, GCP, Azure, and OCI displayed side by side so that I can identify the lowest-cost option.

### Story 3
As a user, I want to see match quality so that I can understand whether the instance is exact, approximate, or overprovisioned.

## Epic: Pricing Data

### Story 4
As a developer, I want seed pricing data so the MVP can work before live API integrations.

### Story 5
As a developer, I want normalized fields so cloud provider data can be compared consistently.
