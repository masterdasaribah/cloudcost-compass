# Demo Script

## Opening

CloudCost Compass is a multi-cloud FinOps MVP that simplifies compute pricing comparison across AWS, GCP, Azure, and OCI.

## Demo Flow

1. Open the app.
2. Select region.
3. Enter 4 vCPU and 16 GB RAM.
4. Select Linux and 730 hours.
5. Run comparison.
6. Review the ranked CSP results.
7. Explain match quality and roadmap.

## Interview Positioning

This project demonstrates product ownership, cloud architecture, FinOps, API design, Agile delivery, and full-stack MVP execution.
